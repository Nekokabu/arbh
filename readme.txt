●はじめに

  このプログラムは、Wiikey baseに付属するDVDDUMPを補助し、バックアップの作成をす
　ることが難しいAction ReplayやFreeloaderなどのディスクのバックアップをより簡単
　にすることができるプログラムです。



●コマンドの説明

　コマンドプロンプトから、cdコマンドなどでプログラムのあるディレクトリへ移動し、
　以下の書式でプログラムを実行してください。

	arbh {option} {driveletter/filename}...

　各オプションの意味は以下のようになります。

	arbh -p F:
		DVDDUMPでフォーマットしたSDカードにARをバックアップするための
		パッチを当てます。


	arbh -s F: OUTPUTFILE.bin
		吸い出したARのデータをファイルに保存します。


	arbh -gcm INPUTFILE1.bin INPUTFILE2.bin OUTPUTFILE.gcm
		吸い出したARのデータからGCMイメージを生成します。



●使い方

　1.まず最初にWiikey baseに付属するDVDDUMPを使い、使用するSDカードをフォーマット
　　してください。次に本ソフトの"-p"オプションを使い、パッチを適用します。
　　　例: arbh -p E:

　2.Wii/GCでDVDDUMPを実行し、ARなどのディスクをエラーが出るまでSDカードに保存します。
　
　3.SDカードをパソコンに接続し、"-s"オプションを使い吸い出したデータをファイルに
　　保存します。
　　　例: arbh -s E: ar0.bin

　4.もう一度DVDDUMPを実行し、吸出しを実行してすぐXボタンを押し続けると次の吸出し
　　位置に移動でできるのでまたエラーが出るまでSDカードに保存します。

　5.3番と同様に吸い出したデータをファイルに保存します。
　　　例: arbh -s E: ar1.bin

　6.吸い出したファイルを元に"-gcm"オプションを使いGCMイメージを生成します。
　　　例: arbh -gcm ar0.bin ar1.bin actionreplay.gcm
